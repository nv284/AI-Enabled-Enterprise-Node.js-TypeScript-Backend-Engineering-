# Module 01 — Exercises

Starter files end in `.starter.ts`; reference solutions end in `.solution.ts`.
**Try the starter yourself first**, then compare.

Run any exercise directly with `ts-node`:

```powershell
npx ts-node exercises/pageviews.starter.ts
npx ts-node exercises/otp.solution.ts
npx ts-node exercises/recent.solution.ts
```

| # | File                         | What you build                       |
|---|------------------------------|--------------------------------------|
| 1 | `pageviews.starter.ts`       | INCR counter + ZINCRBY top-N index   |
| 2 | `otp.solution.ts`            | 60s OTP with single-use verify       |
| 3 | `recent.solution.ts`         | Deduped last-5 search list per user  |

Full requirements are in [../README.md](../README.md#16-exercises-25-min).
