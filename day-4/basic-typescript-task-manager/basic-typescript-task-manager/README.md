# Basic TypeScript Task Manager

## Problem (Traditional Approach)
Many small teams track daily tasks using paper or spreadsheets.
Problems:
- Tasks get lost.
- No consistent status.
- Duplicate entries.
- Difficult to search.
- No reusable code.

## Solution
A very basic TypeScript console application that stores tasks in memory.

This project intentionally avoids:
- High performance optimizations
- Real-time communication
- Databases
- APIs
- Frameworks

Run:
npm install
npm run build
npm start
