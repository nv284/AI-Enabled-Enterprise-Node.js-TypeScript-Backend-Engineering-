# GraphQL Book API

Production-shaped Apollo Server + TypeScript training demo covering:

- **GraphQL schema design** (enums, input types, paginated connections)
- **Resolvers** (queries, mutations, field resolvers, `GraphQLError` handling)
- **Query optimization** — solving the N+1 problem with `DataLoader`
- **AI-assisted development** — prompts for schema evolution and optimization

## Quick start

```powershell
npm install
npm run dev
```

Then open <http://localhost:4000/> for the Apollo Sandbox.
